## 📋 Sobre

Script Python que busca contatos no **Supabase** e dispara mensagens personalizadas via **WhatsApp** usando a **Z-API**. Arquitetura em camadas com separação entre `repository` (acesso ao banco) e `service` (envio de mensagens).

## 🗄️ Setup do banco de dados (Supabase)

Crie um projeto no [Supabase](https://supabase.com) e execute o seguinte SQL no **SQL Editor** para criar a tabela de contatos:

```sql
CREATE TABLE contatos (
  id            BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  nome_completo TEXT NOT NULL,
  numero        TEXT NOT NULL
);
```

O número deve estar no formato internacional sem símbolos, ex: `5521999999999`.

## ⚙️ Variáveis de ambiente

Crie um arquivo `.env` na raiz do projeto com as seguintes variáveis:

```env
# Supabase
SUPABASE_URL=https://xxxxxxxxxxxxxxxxxxxx.supabase.co
SUPABASE_KEY=sua_service_role_key_aqui

# Z-API
ZAPI_INSTANCE_ID=seu_instance_id
ZAPI_INSTANCE_TOKEN=seu_instance_token
```

> As credenciais do Supabase estão em **Project Settings → API**. As da Z-API estão no painel da sua instância em [z-api.io](https://z-api.io).

## ▶️ Como rodar

```bash
# 1. Clone o repositório
git clone https://github.com/lucasvargass/desafio-tecnico-estagio-b2bflow.git
cd desafio-tecnico-estagio-b2bflow

# 2. Crie e ative o ambiente virtual
python -m venv .venv

# Linux/Mac
source .venv/bin/activate

# Windows
.venv\Scripts\activate

# 3. Instale as dependências
pip install -r requirements.txt

# 4. Configure o arquivo .env (veja seção acima)

# 5. Execute
python main.py
```

O script buscará os 3 primeiros contatos da tabela e enviará uma mensagem personalizada para cada um via WhatsApp.

## 📁 Estrutura

```

├── services/
│   └── whatsapp_service.py        # Envio de mensagens via Z-API
├── requirements.txt
└── .env                           # NÃO versionar — já no .gitignore
```
